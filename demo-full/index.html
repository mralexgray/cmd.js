Loading multiple JavaScript files

Inline JS:

var result = "";

getScript([
  "script-one.js",
  "script-two.js",
  "script-three.js"
], 
"script-four.js",
"script-five.js",
[
  "script-six.js",
  "script-seven.js",
  "script-eight.js"
],
"script-nine.js",
[
  "script-ten.js"
], function(success){
  if (success === true) {
    $(".result").append(result);
  }
});

script-x.js contents:

result += "<p>script x loaded</p>";

(Note: script-six.js contains a large commented-out chunk of data to make it a large file. script-seven.js and script-eight.js load before it, as they're in the same array, but later files wait until all three files have been downloaded.)
